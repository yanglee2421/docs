# 数据类型

## 基本类型

1. `undefined`
2. `Number`
3. `String`
4. `Boolean`
5. `Symbol`
6. `BigInt`
7. `null`

## 引用类型

```js
new Object();
new Function();
new Array();
new Promise();
new RegExp();
new Error();
new Set();
new Map();
```

## 声明

|   关键字   |  变量提升  | 块级作用域 | 重复声明 | 可以改 |
| :--------: | :--------: | :--------: | :------: | :----: |
|   `var`    | 全局的顶部 |     无     |   允许   |  可以  |
| `function` |  块的顶部  |     无     |   允许   |  可以  |
|   `let`    |    没有    |     有     |  不允许  |  可以  |
|  `const`   |    没有    |     有     |  不允许  | 不可以 |
