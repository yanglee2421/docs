# npm
~~~powershell
npm config set registry https://registry.npmjs.org
npm config set registry https://registry.npm.taobao.org
npm i -g @vue/cli
~~~
## 初始化
- `npm init`
## 安装
- `npm i`
- `npm i -D`
- `npm i -g`
## 卸载
- `npm uninstall`
- `npm uninstall -g`
## 查看包版本
- `npm view package versions`
