# 指令
- `@import`
~~~scss
@import "@/assets/scss/index.scss";
// 不能直接导入css
~~~
- `@mixin、@include`
~~~scss
@mixin mixin($bg:#000) {
    background:$bg;
}
div {
    @include mixin(#fff);
}
~~~
- `@extend`
~~~scss
.border {
    border:1px red solid;
}
div {
    @extend.border;
}
~~~
- `@media`
~~~scss
div {
    @media (min-width:700px) and (max-width:1100px) {}
    @media (any-hover:none) {}
    @media (any-hover:hover) {}
    @media (any-pointer:none) {}
    @media (any-pointer:coarse) {}
    @media (any-pointer:fine) {}
}
~~~
- `@if、@else if、@else`
~~~scss
$var:1;
@if($var==1){
    $let:1;
}
@else if($var==2){
    $let:2;
}
@else {
    $let:3;
}
~~~
- `@for`
~~~scss
@for $i from 1 to 100 {
    // 1-99
}
@for $i from 1 through 100 {
    // 1-100
}
~~~
- `@each`
~~~scss
$arr:1,2,3;
@each $item in $arr {
    $index:index($arr,$item)
}
~~~
- `@while`
~~~scss
$i:1;
@while ($i<5) {
    $i:$i + 1;
}
~~~
