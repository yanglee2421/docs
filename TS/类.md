# 类

## 基础

```ts
class Person {
  name: string;
  age: number;
  constructor(name: string, age: number) {
    this.name = name;
    this.age = age;
  }
  callName() {
    console.log(this.name);
  }
}
```

## 继承

```ts
class Animal {
  constructor(age) {
    this.age = age;
  }
  run() {}
}
class Cat extends Animal {
  constructor(name: string, age: number) {
    super(age);
    this.name = name;
  }
  callName() {}
}
```

## 抽象类

```ts
/**
 * 只能被继承，不能被实例
 */
abstract class Human {}
class Person extends Human {}

```
