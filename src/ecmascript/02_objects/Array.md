# Array

## 构造器

```js
const arr = new Array();
```

## 静态方法

- isArray
- from

```js
const set = new Set();
Array.isArray(set); //returns false
Array.from(set); //returns []
```

## 实例属性

- length

```js
const arr = [];
arr.length; //returns 0
```
