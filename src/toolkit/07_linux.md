# Linux

## WSL

以下命令依次为：

1. 安装分发版
2. 查看状态
3. 列出已安装的分发版
4. 注销已安装的分发版

```powershell
wsl --install
wsl --status
wsl --list
wsl --unregister ubuntu
```

## IP Forward

```bash
sysctl -w net.ipv4.ip_forward=1
```
