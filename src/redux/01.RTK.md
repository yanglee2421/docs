# RTK

## 安装

```powershell
yarn add react-redux @reduxjs/toolkit
```

## 部署

```jsx
import { Provider } from "react-redux";
import store from "./store";
const dom = document.getElementById("root");
ReactDOM.createRoot(dom).render(
  <Provider store={store}>
    <App />
  </Provider>
);
```
